POST /api/leaderboard

Submit a new score from your Python aim trainer. All fields except mode are required.

// Request body (JSON)
{
  "player":      "xX_Sniper_Xx",
  "score":       98420,
  "accuracy":    94.7,      // 0–100 float
  "reaction_ms": 187,        // avg ms per hit
  "mode":        "timed"      // optional tag: timed or lives
}

// Response 201 Created
{
  "status": "ok",
  "id": "new-uuid",
  "rank": 3
}
GET /api/leaderboard/:id

Get a single entry by its ID.

DELETE /api/leaderboard/:id

Remove an entry. In production, secure this with an admin API key in the X-API-Key header.

Python example — submitting a score after a game session:

# pip install requests
import requests

LEADERBOARD_URL = "https://your-domain.com/api/leaderboard"

def submit_score(player, score, accuracy, reaction_ms, mode="timed"):
    payload = {
        "player": player,
        "score": score,
        "accuracy": round(accuracy, 2),
        "reaction_ms": int(reaction_ms),
        "mode": mode,
    }
    res = requests.post(LEADERBOARD_URL, json=payload, timeout=5)
    res.raise_for_status()
    data = res.json()
    print(f"Score submitted! You ranked #{data['rank']}")
    return data
